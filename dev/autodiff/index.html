Automatic differentiation

There are two ways to compute the gradient of an einsum expression. The first one is to use the OMEinsum package, which is a custom implementation of the reverse-mode automatic differentiation. The second one is to use the Zygote package, which is a source-to-source automatic differentiation tool.

Built-in automatic differentiation

The OMEinsum package provides a built-in function cost_and_gradient to compute the cost and the gradient of an einsum expression.

julia> using OMEinsum  # the 1st way
julia> A, B, C = randn(2, 3), randn(3, 4), randn(4, 2);
julia> y, g = cost_and_gradient(ein"(ij, jk), ki->", (A, B, C))(fill(-3.872779790920286), Any[[2.4605561143711325 -1.0412735134066269 -2.2459958810892053; -0.41868967311449773 0.12610579235440095 0.9260541140436954], [-0.58505816481579 0.3624275395872132 0.5462154957974947 -0.32508229556028134; 1.0414289677363826 0.27873213396852786 -1.3202465250240367 0.6238402941330503; 0.34816824236752325 -0.31978883083161813 -0.28584244337617126 0.1883653781589636], [-0.7355010815367781 0.7488327305422497; 0.8103704960392799 0.027333394536448615; 1.2709576579897202 1.399599253851184; -3.4587251917746675 -3.1486851164597445]])

This built-in automatic differentiation is designed for tensor contractions and is more efficient than the general-purpose automatic differentiation tools.

Using Zygote

The backward rule for the basic einsum operation is ported to the ChainRulesCore, which is used by the Zygote package. Zygote is a source-to-source automatic differentiation tool that can be used to compute the gradient of an einsum expression. It is more general and can be used for any Julia code.

julia> using Zygote  # the 2nd way
julia> Zygote.gradient((A, B, C)->ein"(ij, jk), ki->"(A, B, C)[], A, B, C)([2.4605561143711325 -1.0412735134066269 -2.2459958810892053; -0.41868967311449773 0.12610579235440095 0.9260541140436954], [-0.58505816481579 0.3624275395872132 0.5462154957974947 -0.32508229556028134; 1.0414289677363826 0.27873213396852786 -1.3202465250240367 0.6238402941330503; 0.34816824236752325 -0.31978883083161813 -0.28584244337617126 0.1883653781589636], [-0.7355010815367781 0.7488327305422497; 0.8103704960392799 0.027333394536448615; 1.2709576579897202 1.399599253851184; -3.4587251917746675 -3.1486851164597445])
